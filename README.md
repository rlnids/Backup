# 🧰 Proyecto: Backup y Recuperación en Kali Linux

Este repositorio documenta un trabajo práctico sobre **backups y recuperación de sistemas** realizado en Kali Linux. Se incluyen comandos manuales, automatización con scripts, verificación de integridad, simulación de desastres y un plan de recuperación detallado.

## 📌 Objetivo

Demostrar habilidades en:
- Creación de backups completos e incrementales con `tar`.
- Automatización mediante scripts en Bash.
- Verificación de integridad de backups.
- Simulación de pérdida de datos y recuperación.
- Monitoreo del sistema (logs, procesos, conexiones).
- Contención de incidentes (servicios web y BD).

## 📁 Estructura del repositorio

- `backup_enterprise.sh` → Script principal (no subido por privacidad, explicado en la documentación).
- `docs/` → Documentación detallada de cada fase.
- `logs/` → Ejemplo de archivo de log generado por el script.
- `.gitignore` → Archivos excluidos del repositorio.

## 🚀 Cómo usar este proyecto

1. Clona el repositorio.
2. Revisa la documentación en `docs/` para entender cada comando y procedimiento.
3. Si deseas probar el script, crea tu propia versión (el original no está incluido).
4. Sigue el plan de recuperación para simular un desastre.
